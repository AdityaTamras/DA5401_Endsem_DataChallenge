# DA5401_Endsem_DataChallenge
DA5401 : Data Analytics Lab : End-Semester Data Challenge

The repository contains the relevant code and report for the DA5401 : Data Analytics Lab End-Semester Data Challenge. The aim of the challenge is to develop a metric learning model to predict the "fitness" or similarity score between a specific AI Evaluation Metric Definition and a corresponding Prompt-Response Text Pair. The DA5401_Endsem.ipynb contains the code file containing the relevant plots and analysis. The observation and conclusions are mentioned in the report.
